# Prisoners — Дилемма заключённого (8-bit симуляция)

## Описание проекта
Веб-страница с 8-битной графикой (стиль NES/Dendy), симулирующая две стратегии решения Дилеммы заключённого. После анимации показываются статистические данные.

**Цель:** наглядно показать разницу между стратегиями через пиксельную анимацию.

## Архитектура

- Единственный файл `index.html` — без npm, без бэкенда, без сборщиков
- Вся логика, стили и спрайты — **только** в одном HTML-файле
- Внешние ресурсы (шрифты, библиотеки) разрешены только через CDN-ссылки в `<link>` или `<script>`

## Графика и анимация

- **8-битный стиль:** вся графика рисуется через Canvas 2D API с пиксельной отрисовкой спрайтов
- **Анимация:** пошаговая с использованием `requestAnimationFrame`, стиль 8-битных RPG (покадровая смена спрайтов, перемещение персонажей по тайловой сетке)
- **Палитра:** ограниченная (8–16 цветов), характерная для NES
- **Разрешение канваса:** низкое (например, 256×240) с CSS-масштабированием для чётких пикселей (`image-rendering: pixelated`)

## Стратегии

Две стратегии поведения заключённых, реализованные в JS-логике:

1. **Стратегия A** — описывается в интерфейсе
2. **Стратегия B** — описывается в интерфейсе

Анимация показывает N раундов взаимодействия. По завершении — экран со статистикой.

## Локализация

- Интерфейс на **русском** и **английском**
- Механизм: JS-объект с ключами (`ru` / `en`), переключение через кнопку/селектор в UI
- Тексты на экране статистики, кнопки, описания стратегий — всё через этот объект

## Взаимодействие с пользователем

1. Выбор языка (RU/EN)
2. Выбор стратегии (2 варианта)
3. Кнопка **«Старт»** — запускает анимацию
4. Опционально: чекбокс «Показывать текст»/«Show text» — включает текстовые подписи хода в анимации

## Технические требования

- HTML5 + CSS3 + Vanilla JS (ES6+)
- Canvas 2D API для отрисовки
- `@font-face` для пиксельного шрифта (можно через CDN, например, Press Start 2P)
- Никаких зависимостей — только то, что подключается через `<script src="...">` или `<link href="...">`
- Файл должен открываться **из коробки**: `open index.html` в браузере

## Структура (единственный файл)

```
index.html
├── <head>
│   ├── pixel font (CDN)
│   └── styles (inline <style>)
├── <body>
│   └── HTML-разметка и inline <script>
└── все скрипты внутри:
    ├── i18n (ru/en объект)
    ├── стратегии (функции выбора)
    ├── спрайты (определение пиксельных данных)
    ├── рендер (Canvas рисование)
    └── анимация (game loop через requestAnimationFrame)
```
