## 📖 Что это?

Веб-страница с **8-битной графикой** в стиле NES/Dendy, которая визуально симулирует классическую **Дилемму заключённого (100 prisoners problem)**.

После анимации показывается статистика: сколько заключённых нашли свой номер, какой процент успеха.

> **Живое демо:** открыть `index.html` в любом современном браузере.

## 🧠 Теория: Дилемма заключённого (100 prisoners problem)

Классическая задача теории вероятностей:

> **Условие:**
> 100 заключённых, 100 коробок. В каждой коробке — номер одного из заключённых (случайное расположение).  
> Заключённые заходят в комнату по одному и могут открыть до **50 коробок**. После выхода они не могут общаться с теми, кто ещё не заходил.  
> Если **все 100** нашли свой номер — всех отпускают. Если хотя бы один не нашёл — казнят всех.

Наивная вероятность успеха при случайном выборе — **(½)¹⁰⁰ ≈ 0.00000000000000000000000000008** (практически ноль).

**Оптимальная стратегия** (циклический обход) даёт вероятность успеха **~31%**.

---

## 🎮 Две стратегии в проекте

### 1️⃣ Случайная (RANDOM)
Каждый заключённый открывает коробки **наугад**, не запоминая результаты.  
50 попыток, случайный выбор на каждом шаге.  

**Результат:** почти гарантированный провал (один из 100 почти наверняка не успеет).

### 2️⃣ Циклы (CYCLE)
Каждый заключённый:
1. Начинает с коробки, **номер которой совпадает с его собственным номером**
2. Открывает её, смотрит номер внутри
3. Идёт к коробке с этим номером
4. Повторяет, пока не найдёт свой номер (или не кончатся 50 попыток)

**Почему это работает:** номера образуют циклы в перестановке. Заключённый гарантированно пройдёт по циклу, который содержит его номер. Если длина цикла ≤ 50 — он успеет. Вероятность, что в случайной перестановке из 100 элементов нет циклов длиннее 50 — **~31%**.

### 🌐 `index.html` (единственный файл приложения)

| Компонент       | Описание |
|----------------|----------|
| Canvas 2D API  | Пиксельная отрисовка спрайтов в стиле 8-bit |
| i18n           | Русский / English — объект с ключами, переключение кнопками |
| Стратегии      | JS-функции `runRandom()` и `runCycle()` |
| Анимация       | `requestAnimationFrame`, пошаговое перемещение по тайловой сетке |
| Статистика     | Экран результатов после каждого прогона |

### ☕ Java / Gradle

Java-модуль (`src/`) — минимальная заготовка с `Main.java`. Фактическая симуляция работает в браузере через `index.html`. Gradle-обёртка позволяет удобно открывать проект в IntelliJ IDEA.
